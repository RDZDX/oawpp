Oawpp - Open app with parameter primitive

Basic app to launch vxp with parameter for MRE platform mobile phone (including Nokia S30+). Tested on Nokia 225 with display resolution 240x320. How to use: first step select text file or input text to editor window, second step select vxp app file. For using with Nokia mobile phone, app must be signed with IMSI (your SIM card) code. https://vxpatch.luxferre.top. Application files - "Oawpp.vxp"
